# Neovim Configuration

My custom Neovim, not perfect. The way i like it

---
## Quick Installation

Backup your current config and clone this repository into the standard Neovim configuration path:
```Bash
mv ~/.config/nvim ~/.config/nvim.bak 2>/dev/null
git clone https://github.com/Ruohao1/neovim-configuration.git ~/.config/nvim
nvim
```
